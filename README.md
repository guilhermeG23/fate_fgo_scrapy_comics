### Download dos comics

Buscar todos os comics do **Fate FGO** via **Python**, **OBS:** Fiz isso em um virtualenv;

#### Requisitos 
* Instale o requeriments.txt e execute o programa; 

#### Orientação

Quando executar o programa, lembra-se que tem duas opções:

* BOOK Colorido:
```
python <programa>
```

* BOOK escala de cinza
```
python <programa> 1
```
